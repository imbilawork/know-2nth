2nth.ai · About this dashboard

Same data. Same system. Different job.

This workshop view and the customer storefront are two front-ends talking to the same private ERPNext instance running on GCP in africa-south1. Every number on this page — stock levels, work-order progress, AR aging, the schedule — is live operational data. When a customer accepts a quote on the storefront, an Order appears here. When a workshop manager marks a Work Order complete, the customer's order tracker updates.

ERPNext is operationally brilliant and visually brutal. The 2nth pattern is to keep ERPNext as the system of record — for everything the SARS auditor wants to see — and wrap it in views that are built for the job: fast for ops, beautiful for customers, minimal for exec.
